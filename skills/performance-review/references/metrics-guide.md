# Twitter 关键指标定义与基准

## 核心指标定义

### 互动率 (Engagement Rate)
- **公式**：(点赞 + 转发 + 评论 + 点击) / 印象数 × 100%
- **行业基准**：
  - 优秀：> 5%
  - 良好：3-5%
  - 一般：1-3%
  - 需改进：< 1%
- **加密领域特点**：因为话题争议性强，加密KOL互动率通常偏高

### 粉丝增长率
- **公式**：本期净增粉丝 / 期初粉丝数 × 100%
- **健康增长**：周增长 1-3%（取决于基数）
- **爆发增长**：单条爆款可能带来 5-10% 增长

### 转化率（Profile → Follow）
- **公式**：新增粉丝 / 主页访问数 × 100%
- **基准**：10-20% 算健康
- **提升方法**：优化Bio、置顶推文、头像

### 推文存活时间
- **定义**：一条推文获得 80% 互动的时间窗口
- **Twitter平均**：4-6小时
- **线程**：24-48小时（因为持续被回复）

## 内容分类标准

| 分类 | 判断标准 |
|------|---------|
| 锐评快评 | 单条推文，强观点，< 200字 |
| 数据分析 | 包含具体数字/图表 |
| 长线程 | 3条以上连续推文 |
| 快讯转评 | 引用/转推 + 评论 |
| meme/段子 | 幽默/讽刺类内容 |
| 互动帖 | 提问/投票/征集类 |

## 数据采集方法

### 通过 Twitter API v2
```
# 获取用户推文指标
GET /2/users/:id/tweets?tweet.fields=public_metrics,created_at

# 返回字段
{
  "public_metrics": {
    "retweet_count": X,
    "reply_count": X,
    "like_count": X,
    "quote_count": X,
    "impression_count": X
  }
}
```

### 手动追踪（无API时）
每日记录到表格：
| 日期 | 推文内容 | 类型 | ❤️ | 🔄 | 💬 | 印象 | 互动率 |

## 周报/月报模板

### 周报结构
1. 本周概览（3个核心数字）
2. Top 3 推文分析
3. Bottom 3 推文分析
4. 与上周对比
5. 下周优化建议

### 月报结构
1. 月度概览（增长/互动/内容）
2. 内容策略效果评估
3. 粉丝画像变化
4. 竞对对比
5. 下月策略调整
