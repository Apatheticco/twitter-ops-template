# 回复策略模板库

## 场景化回复模板

### 1. 被质疑数据准确性
```
"感谢指正。数据来源是[来源]，我再核实一下。
如果有更新会第一时间修正。
不过核心观点[观点]不受这个数据点影响——因为[逻辑]。"
```

### 2. 被说是马后炮
```
"理解。回看确实像马后炮，但当时的判断逻辑是：[逻辑链]。
更重要的是现在怎么看：[当前观点]。
市场永远是对的，我只是提供一个分析框架。"
```

### 3. 有人问操作建议
```
"不做投资建议。但从数据角度看：
- [事实1]
- [事实2]
风险自担，DYOR。"
```

### 4. KOL发了你不同意的观点
```
"有意思的角度。不过还有一个维度值得考虑：[你的数据/逻辑]。
从[具体指标]来看，情况可能更复杂。
两种可能都存在，关键看[什么条件触发]。"
```

### 5. 新粉丝提基础问题
```
"好问题，简单说：[简洁回答]。
如果想深入了解，建议看[资源]。
之前也写过一个线程，搜[关键词]可以找到。"
```

### 6. 被无理攻击
```
[不回复，或者简短一句]
"数据摆在那儿，各人判断。"
[如果持续骚扰，直接mute/block]
```

## 主动评论模板

### 在KOL帖子下发评论
```
"补充一个数据点：[数据]。
从这个角度看，[你的独特观点]。
[可选：@其他相关KOL，形成三方讨论]"
```

### 在热门话题下抢占位置
```
"大家都在讨论[话题]，但忽略了一个关键变量：[变量]。
[1-2句数据支撑]。
这才是决定走势的核心。"
```

## 互动节奏指南

| 时段 | 动作 | 时长 |
|------|------|------|
| 早间(9:00) | 检查过夜互动，回复质疑 | 15分钟 |
| 午间(12:00) | 主动在3-5个KOL帖子下评论 | 10分钟 |
| 晚间(20:00) | 回复当日评论，参与热门讨论 | 15分钟 |
