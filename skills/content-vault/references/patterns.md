## 通用高表现模式（可参考）

### 模式 1：BREAKING 紧急感
- **结构**：🚨/BREAKING → [核心信息] → [影响判断]
- **适用**：重大突发事件、数据发布
- **关键**：速度 > 完美

### 模式 2：口语情绪爆发
- **结构**：[情绪词开头] → [事件+数据] → [口语化总结]
- **适用**：令人震惊的数据、反直觉事件
- **关键**：真实情绪，不要表演

### 模式 3：嘲讽冷评
- **结构**：[短句定性] → [事件描述] → [冷嘲热讽点评]
- **适用**：行业闹剧、打脸事件
- **关键**：简短有力，4-8字的定性最佳

### 模式 4：人物冲突框架
- **结构**：[A vs B] → [冲突背景] → [双方观点] → [开放性提问]
- **适用**：KOL互怼、行业争议
- **关键**：呈现双方立场，引导站队讨论

### 模式 5：数据排行/对比
- **结构**：[对比框架] → [数据列表] → [一句话总结]
- **适用**：周期对比、排名变化、链上数据
- **关键**：数据可视化，有对比才有冲击

### 模式 6：升级版日报
- **结构**：[情绪Hook开头] → [编号信息列表，每条带点评] → [互动结尾]
- **适用**：每日固定栏目
- **关键**：每条信息后必须加点评，结尾必须有互动钩子

## 模式快速匹配表

| 场景 | 推荐模式 | 格式 |
|------|---------|------|
| 重大突发 | 模式1 BREAKING | 单推 |
| 数据爆冷/爆热 | 模式1 + 模式2 | 单推 |
| 行业闹剧 | 模式3 嘲讽 | 单推 |
| KOL互怼 | 模式4 人物冲突 | 单推 |
| 周期数据对比 | 模式5 数据排行 | 单推/Thread |
| 每日汇总 | 模式6 升级日报 | 单推 |
