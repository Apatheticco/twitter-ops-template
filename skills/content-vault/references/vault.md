# 📦 推文素材库 Content Vault

> ⚠️ 初始化模板 — 运行 content-vault 扫描历史推文后自动生成

## 使用方法

1. 告诉 Claude："帮我建立素材库" 或 "扫描我的历史推文"
2. Claude 会通过 MCP 拉取你的历史推文数据
3. 按 ER（互动率）自动筛选 S/A 级推文
4. 拆解每条高表现推文的成功元素
5. 生成本文件

## 筛选标准

```
互动率 = (点赞 + 转发 + 评论 + 引用) / 浏览数

🏆 S级：ER Top 10%（且浏览 ≥ 500）
⭐ A级：ER Top 25%
📌 观察：ER 高于均值 1.5 倍
```
