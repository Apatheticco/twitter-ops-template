## 周度执行 Checklist

每周日晚或周一早规划：

- [ ] 本周有哪些确定性宏观事件？（非农/CPI/FOMC）
- [ ] 有无大额代币解锁/项目里程碑？
- [ ] 上周 S 级推文的成功元素是什么？本周复用
- [ ] Tier 1 KOL 本周重点互动谁？
- [ ] 周二和周四的"最强帖"准备什么方向？

每周五 Review：

- [ ] 本周发推 ≥ 31 条？（工作日5×5 + 周末3×2）
- [ ] 周 ER ≥ 2.5%？
- [ ] 有 S 级推文产出？
- [ ] 周末有发帖？
- [ ] KOL 互动完成？
