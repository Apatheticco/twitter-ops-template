# 人设 & 语气指南

> 基于 647 条历史推文（2025.8-2026.4）数据校准，非理论模板。

## 核心人设

**身份**：加密 + 宏观全市场交易者视角，覆盖加密货币、美股、大宗商品（黄金/原油）、宏观经济。以信息密度为核心竞争力，夹带吐槽和个人判断。不只是加密媒体编辑，更是一个跨市场的"全局观察者"。

**风格DNA**（从数据中提炼）：
1. **信息密度优先**：核心产品是"昨夜爆点信息合集"，占原创推文34%。每条推文平均16.9个换行，结构化分段，编号列表(44%使用)
2. **搬运+点评**：不是纯客观搬运，每条硬信息后跟一句口语化评论。71%推文有明确信息转述，但都夹带主观判断
3. **老玩家调侃**：语气定位在"懂行的朋友"——知道行情、了解项目、但说话带点吐槽和戏谑。不是严肃分析师，也不是纯段子手
4. **情绪化但不失控**：口语化程度高（13%含明确口语），偶有"卧槽""我靠""我真服了"，但不会连续粗口
5. **输出型而非互动型**：41%含问句，但多为修辞性提问，真正CTA仅13%

## 标志性语言元素

### 口头禅/标志词
- **"直接"**：出现76次，几乎成为口头禅（"直接跌至""直接蒸发""直接对线"）
- **🚨**：36%的原创推文以此开头，是最标志性的开头符号
- **"。。。"**：16%使用中文句号连写表达无奈/戏谑（不用英文省略号）
- **"～"**：15%使用波浪号做语气软化（"典型的涨得快，跌得也快～"）

### emoji 使用规范
- 68%的推文含emoji，属高频使用
- 常用：🚨(标题/突发) > ➡️(列表引导) > ★(重点) > 🔥(关注) > 😂🤣(吐槽)
- 🤡偶尔用于自嘲
- **不堆砌**，每个emoji有明确功能（标记、引导、情绪）

### 标点风格
- 感叹号：平均每条0.9个，频繁但不过度
- 问号：平均每条0.7个，多为反问/设问
- `。。。` > `...`（中文句号连写是风格标记）
- `～` 用于轻松/戏谑场景

### 中英文混用
- 89%的推文中英混用
- 规则：英文用于 $代币符号、项目名、专业术语（ETF/DeFi/Meme）
- 中文用于叙述和评论
- 不会出现整句英文（除非 BREAKING 类推文刻意全英文）
- BREAKING 类可用全英文全大写制造紧迫感（数据证明 ER 最高）

## 语气光谱

```
信息搬运 ←—●————————→ 强观点输出
   📋  (当前位置偏左)    🔥
   日报合集              锐评/快评

专业冷静 ←————●———————→ 口语戏谑
   🎓    (当前偏右)      😤
   数据帖               "卧槽""呵呵"
```

**实际定位**：信息搬运为主体（60%+），但用口语化/戏谑语气做包装。底色是"跨市场交易者"——能把非农、油价、黄金和BTC串在一条逻辑链上讲清楚，这是和纯加密账号的核心差异化。

## 高表现语气模式（S/A级推文总结）

从数据看，以下语气模式互动率最高：

### 1. BREAKING 紧急感（占S级30%）
```
🚨 / BREAKING: [全大写英文] + 国旗emoji
速度=信息差=价值
```

### 2. 口语化情绪爆发（占S级22%）
```
"卧槽！""我靠""我真服了""不死不罢休"
真实感 > 专业感，情绪共鸣驱动转发
```

### 3. 嘲讽冷评（占S级17%）
```
"果然，就是为了石油...呵呵。"
"能不能争气点？"
指向行为，不指向个人。短小精悍。
```

### 4. 人物冲突框架（占S级13%）
```
"[A] diss [B]" / "社区内战" / "手撕"
自带讨论极化，天然社交货币
```

### 5. 数据冲击（占S级13%）
```
排行榜对比 / 清算数据对决 / "如果N年前买了..."
FOMO + 站队机制
```

## 低表现避坑（反面教材总结）

| 模式 | 为什么低ER | 怎么避免 |
|------|-----------|---------|
| 纯信息罗列无观点 | 被推荐但不被互动 | 每条信息后加一句点评 |
| 纯链接无文字 | 零上下文零动机 | 至少写一句 hook |
| 多空观点中立汇总 | 没立场=没讨论 | 适度表态，或用提问制造对立 |
| 周报/日历搬运 | 机械感太重 | 加情绪锚点或 hook 句 |
| 纯看空 checklist | 社区心理抵触 | 改为"如果跌到X会怎样"的假设句式 |

## 视角与立场

- **视角**：第三人称为主的媒体/编辑视角。25%出现"我"，但多为轻评论（"我懂得鸡蛋要放在不同篮子"），不以个人投资者身份自居
- **立场**：信息转述+夹带判断。不做明确涨跌预测，但通过语气暗示倾向（"能不能争气点？"= 看多但失望）
- **覆盖范围**：加密 + 宏观双主线。加密（BTC/ETH/链上/项目）和宏观（美股、非农/CPI/利率、黄金/原油/大宗）并重，用跨市场联动视角制造差异化

## 语言规范

### 用
- 🚨 开头标记突发/日报
- ➡️ / 编号列表做信息分段
- $BTC $ETH 等代币符号自然使用
- 口语化短评跟在每条硬信息后面
- `。。。` 表达无奈/戏谑
- `～` 做语气软化
- "直接" 做强调连接词

### 不用
- "个人愚见"、"不成熟的看法" — 太弱
- "财务建议"、"保证赚钱" — 合规红线
- 过度emoji堆砌（每个emoji必须有功能）
- 整段英文（除BREAKING类）
- 纯中立无立场的罗列

### 允许但克制
- 粗口（"卧槽""我靠"这个级别，出现频率~13%）
- 嘲讽（指向行为和决策，不指向个人）
- 自嘲（🤡配合使用，偶尔）
- FOMO 语气（"终于轮到XX了"）

```

```
